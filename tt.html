1 NAME

1.1 formula

H(X|Z)+H(Y|Z) ≥ H(X,Y|Z)

1.2 proof

If we omits the conditional varible, the inequality is still established. It can easily derived from H(X,Y)=H(X)+H(Y|X).

2 NAME

2.1 formula

H(X,Z)+H(Y,Z) ≥ H(Z)+H(X,Y,Z)

2.2 proof

Subsitute each variable one another, we got three inequality. Then add them up. We got 2(H(X,Y)+H(Y,Z)+H(X,Z)) ≥ 3*H(X,Y,Z) + H(X)+H(Y)+H(Z), which gives an intutively point view of the way working the proof out.

3 Subadditivitty of entropy

3.1 formula

H(X,Y) ≤ H(X)+H(Y)

3.2 proof

I(X;Y) = D(p(x,y)||p(x)p(y)) = H(X)+H(Y)-H(X,Y) ≥ 0

Equivalent with the previous inequality

6 Data-processing inequality

6.1 formula

If X->Y->Z, then I(X;Y) ≥ I(X;Z)

6.2 proof

By the chain rule, we can expand mutual information in two different ways:

  • I(X;Y,Z) = I(X;Z) + I(X;Y|Z)
  • I(X:Y,Z) = I(X;Y) + I(X;Z|Y) Since X and Y are conditionally independent given Y, we have I(X;Z|Y)=0. Since I(X;Y|Z)≥ 0, we have I(X;Y) ≥ I(X;Z) Similarly, one can prove that I(Y;Z) ≥ I(X;Z).

6.3 formula

If X->Y->Z, then I(X;Y|Z) ≥ I(X;Y).

6.4 proof

We note in I(X;Y,Z)=I(X;Z)+I(X;Y|Z)=I(X;Y)+I(X;Z|Y). That I(X;Z|Y)=0, by Markovity, and I(X;Z)≥ 0. Thus, I(X;Y|Z) ≥ I(X;Y).
